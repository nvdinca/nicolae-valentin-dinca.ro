# Contracte

Acest folder conține modele de contracte pentru proiecte de dezvoltare software.

## Conținut

- **model-cesiune-drepturi-autor-software.md** – Model de contract de cesiune a drepturilor de autor pentru programe de calculator. Folosește-l când cedez drepturile asupra software-ului (cod, documentație) către client / afacere. Complectează câmpurile [_____], apoi exportă în Word sau PDF pentru semnare.

## Utilizare

1. Deschide fișierul `.md` și completează toate câmpurile între paranteze.
2. Recomandat: verificare de către un avocat înainte de semnare.
3. Semnează tu (Cedent) și reprezentantul afacerii (Cessionar); păstrează câte un exemplar semnat.
